overwrite

Availability

AS2 and AS3.

Usage

... overwrite:value, ...

Parameters

value:Boolean — Whether or not the new tween will overwrite other activate tweens that are applied to the same object, on the same property, and which overlap the time the new tween will take. Enabling this simply removes the existing tweens. Not enabling this may work, but the two existing tweens will run in parallel - the newest tween taking priority in setting the final property values - and may lead to undesirable results. The default value for this parameter (when omitted) is false.

Examples

// Slides a MovieClip to _x = 100 in 2 seconds (AS2)
Tweener.addTween(myMovieClip, {_x:100, time:2});
// Add a new slide, overwrite the old one
Tweener.addTween(myMovieClip, {_x:200, time:2, overwrite:true});
